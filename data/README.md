\# M5 Dataset



\## Download Instructions



1\. Go to the \[M5 Forecasting Competition on Kaggle](https://www.kaggle.com/c/m5-forecasting-accuracy/data)

2\. Download the following files:

&nbsp;  - `sales\_train\_validation.csv`

&nbsp;  - `calendar.csv`

&nbsp;  - `sell\_prices.csv`

3\. Place them in this `data/` folder



\## Data Description



\- \*\*sales\_train\_validation.csv\*\*: Daily unit sales by item and store (30,490 series)

\- \*\*calendar.csv\*\*: Date information including events and holidays (1,969 days)

\- \*\*sell\_prices.csv\*\*: Weekly selling prices by item and store



Total size: ~200 MB
